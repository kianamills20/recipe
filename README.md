# Herbal Recipes — Frontend

This is a minimal Vite + React frontend scaffold for the Herbal Recipes app.

Quick start:

```bash
cd /Users/kiana/Desktop/coursework/block44/recipe
npm install
npm run dev
```

Open the URL Vite prints (usually http://localhost:5173).

What this includes:
- Basic green/herb aesthetic styles in `src/styles.css`
- Sample health-focused recipes in `src/data/recipes.js`
- Components: `RecipeList` and `RecipeDetail` in `src/components`
